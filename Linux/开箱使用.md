# 新机到手
注:这是以WSL为平台的ubuntu虚拟机
## 设置root密码
直接 sudo passwd root
输入当前用户的密码即可设置root密码

## WSL相关
在使用的时候整天遇到线程被占用的问题
启动时会自动以默认用户（orion）创建后台进程,即使你 sudo 或 su root，orion 的进程依然存在，导致修改失败。
直接关掉重来
1. Windows PowerShell 中彻底关闭 WSL
wsl --shutdown
2. 以 root 身份启动（跳过 orion 登录）
wsl -u root
## 改用户名
跟着上面
1.登录名改掉
sudo usermod -l newuser olduser
2.家目录路径改掉(改家目录并移动内容（-m 表示 move）)
sudo usermod -d /home/ubuntu -m ubuntu
3.改同名用户组（如存在）
groupmod -n ubuntu orion 2>/dev/null

## 虚拟机安装位置和存储空间
安装好虚拟机后
运行以下命令
```
wsl --list --verbose    
用于查看当前虚拟机的用户名
Get-ChildItem HKCU:\Software\Microsoft\Windows\CurrentVersion\Lxss | ForEach-Object {Get-ItemProperty $_.PSPath} | Select DistributionName, BasePath
查看虚拟机存储空间的路径，好像默认的是C盘
```
如果你是在微软商店上下的，那虚拟机数据盘默认绝对不会自己选路径，它默认永远把Ubuntu 本体 + 虚拟机数据（ext4.vhdx）放在C盘
