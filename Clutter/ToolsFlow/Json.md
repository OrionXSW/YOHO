1. 什么是 JSON？
JSON 全称 JavaScript Object Notation（JavaScript 对象表示法），是一种轻量级的纯文本数据交换格式。它有两个核心特点：
- 对人友好：结构清晰，一目了然。
- 对机器友好：几乎所有编程语言都能轻松解析和生成它。
今天，JSON 是 Web API、配置文件、日志数据等领域最主流的格式，已经完全取代了早期的 XML。
目前接触到JSON的地方是在调用API获取天气信息，AI api配置，vscode.json配置

## 2. 两个基础结构

| 结构 | 符号 | 说明 |
| --- | --- | --- |
| 对象 | `{}` 花括号 | 无序的键值对集合。键必须是带双引号的字符串。 |
| 数组(Array) | `[]` 方括号 | 有序的值列表。 |
