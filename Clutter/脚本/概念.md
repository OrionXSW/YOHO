# 概念
脚本能把手工重复敲的命令、复杂的操作流程固化成一个可执行文件，一键运行，从而显著提高效率。

Shell 是一个通用术语，指的是在操作系统外层，作为用户和系统内核之间“翻译官”的程序。你把命令告诉它，它再指挥电脑去执行。

需要注意的是，图形化界面（GUI，Graphical User Interface）本质上也是一种 Shell，而我们通常讨论的命令行界面（CLI，Command-Line Interface）则属于 CLI Shell。

shell脚本是通用的，能在不同的系统上运行，而Batch脚本(批处理脚本)只能在Windows上运行
