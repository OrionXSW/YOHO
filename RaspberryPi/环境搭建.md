# 网络
## 联网
1.
```
# 扫描 
sudo wpa_cli -i wlan0 scan # 扫描 
sleep 2 # 等两秒
wpa_cli -i wlan0 scan_results # 显示扫描结果
example wpa_cli -i wlan0 scan_results | grep WDX # 只显示WIFI名里有WDX字符的WIFI

# 自动获取空闲编号，无需手动填0/1/2 
netid=$(wpa_cli -i wlan0 add_network) 
# 填入WiFi名称和密码 
sudo wpa_cli -i wlan0 set_network $netid ssid '"WiFi名字"' 
sudo wpa_cli -i wlan0 set_network $netid psk '"WiFi密码"' 
# 启用并永久保存 
sudo wpa_cli -i wlan0 enable_network $netid 
sudo wpa_cli -i wlan0 save_config

# 切换WIFI
# 把数字换成目标WiFi的id，会自动断开当前WiFi并连接新WiFi 
sudo wpa_cli -i wlan0 select_network 目标编号
```
x.x 查看一共保存的WIFI配置
`sudo wpa_cli -i wlan0 list_networks

0.查看WIFI连接情况
`sudo wpa_cli -i wlan0 status`
注：
```
- `ssid`：当前连接的热点名称（确认是连了 `WDX_WIFI` 还是 `REDMI K80 Ultra`）。
    
- `wpa_state`：`COMPLETED` 代表认证成功已连接；`DISCONNECTED` 或 `SCANNING` 表示断开了。
    
- `ip_address`：当前被分配的 IP 地址（有地址才能通信）。
    
- `freq`：频率（如 `2412` 表示 2.4G，`5200` 表示 5G）
```
