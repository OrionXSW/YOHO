***电容_V1***

电容组成：两个导体中间夹着绝缘材料

电解电容是有正负极的，也有耐压值

# 特性

## 电容两端相对电压不能突变

电容两端的**相对电压**不能突变，只要电容不充电或放电，电容两端的电压就不变
注意：电容两端的相对电压不能突变，但两端电压可以同时突变
场景：电容柜两端电压为0V时，上电一瞬间电容相当于导线，随着时间推移，靠近地那段的电压在不断下降直到零，另一端则在不断往上升直至等于赋予给它的电压值

# 储能特性
电容，相当于一个小电池，其充电速度与电容大小以及充电电流有关
电容的充电时间参数：$$\tau = RC$$
根据$\tau$和电流曲线，可以认为3-5$\tau$后，电容充满电，当然，是大致情况
凭借这个，开发出电容的延时特性
比如：上电延时
首先得确定电容容值，再根据目标延时时间来不断调整电阻阻值

| 符号  | 物理量  | 单位  | 简写  |
| --- | ---- | --- | --- |
| τ   | 时间常数 | 秒   | s   |
| R   | 回路电阻 | 欧姆  | Ω   |
| C   | 电容   | 法拉  | F   |
同样也可以实现关断延时

典型的有复位电路
![[assets/电容_V1/file-20260623200500284.png]]
上电瞬间，**电容两端电压不能突变**，因此 RESET 点的电压**被强制维持在 0V（GND 电位）**。随着时间推移，电源 VCC 通过电阻 R2（10k）给电容 C10（0.1μF）充电，RESET 点电压**按指数曲线规律上升**。当该电压上升到单片机判断为‘高电平’的阈值（通常为 0.7×VCC）时，单片机退出复位状态，完成上电启动。
1.按下按键的瞬间（强制复位）
**电容状态**：按键闭合，相当于用一根阻抗极低（接近0Ω）的导线把电容两端（RESET 和 GND）直接短接。此时电容不再是慢慢放电，而是瞬间（微秒级）放电完毕。
结果：RESET点电压瞬间被强制拉低至 0V，单片机检测到低电平，立刻进入复位状态。只要手指一直按住，电压就一直保持 0V。
2.松开按键的瞬间（重新充电）
**电容状态**：按键断开，VCC3.3 再次通过电阻 R2 给电容 C10 充电。此时**物理过程与上电瞬间完全一致**。    
**结果**：RESET点电压从 0V 开始重新按指数曲线上升，当升到单片机的复位阈值时，单片机退出复位状态，恢复正常运行。

**原理图分析：** RC延时电路，R = 10 K C = 104 => 10 * 10^4 = 100,000pF = 0.1uF = 0.1×10^(-6) 法拉
$$\tau = R * C => 10 * 10^3 * 0.1 * 10^{-6} = 1ms$$
需要注意的是计算出的 **1ms** 只是 RC 电路的**物理时间常数**。单片机（MCU）实际执行完复位的**有效延时时间**并不是 ττ，而是取决于单片机**复位引脚判定为“高电平”的门限电压**（通常是电源电压的 0.5~0.7 倍）。
公式可以算出对应的$\tau$值：
$$V(t) = V_{CC} \times \left(1 - e^{-\frac{t}{\tau}}\right)$$
推导得到：
$$t = -\tau \times \ln\left(1 - \frac{V_{th}}{V_{CC}}\right)$$
- 充电到 **50%**（0.5 VCC）需要：**0.7 τ**
- 充电到 **63.2%**（这是 ττ 的物理定义，1−1/e1−1/e）需要：**1.0 τ**
- 充电到 **70%**（0.7 VCC）需要：**1.2 τ**
- 充电到 **90%**（0.9 VCC，通常认为电容彻底充满）需要：**2.3 τ**
