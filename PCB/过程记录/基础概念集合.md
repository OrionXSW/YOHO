# 灌电流和拉电流
【拉电流 (Source Current)】
- 方向：电流从单片机I/O口流出 → 流向负载 → 到地。
- 条件：I/O口输出高电平（逻辑1）时，作为电流源。
- 传统51能力：很弱（通常 < 1mA），直接驱动LED亮度不足。
- 等效：I/O口“向外拉”电流。

【灌电流 (Sink Current)】
- 方向：电流从外部流入单片机I/O口 → 到地。
- 条件：I/O口输出低电平（逻辑0）时，作为电流阱（吸收电流）。
- 传统51能力：较强（典型10-20mA），可直接点亮普通LED。
- 等效：I/O口“向内灌”电流。

【驱动LED建议（传统51）】
- 采用灌电流方式：LED正极接VCC → 限流电阻 → LED负极接I/O口。
- I/O口输出低电平 → LED点亮。
- 务必串联限流电阻（例如200~1kΩ），防止过流损坏I/O口。

【注意】增强型51（如STC15/STC8）可配置推挽输出，拉、灌能力均可达20mA以上，两种接法皆可，以数据手册为准。
